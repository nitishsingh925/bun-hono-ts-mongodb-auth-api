# bun-hono-ts-mogodb-auth-api

To install dependencies:

```bash
bun install
```

## Environment Variables:

To ensure change file name ".env.sample to `.env`"

To Simple Start:

```bash
bun start
```

To Development Start:

```bash
bun dev
```
